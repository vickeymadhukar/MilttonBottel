# Milton 3D Scroll Landing Page

A beautiful, interactive, AWWWARDS-style 3D scrolling landing page for Milton Bottles.

## Features
- **3D Model Integration**: Uses Three.js and GLTFLoader to render a 3D `.glb` model directly in the browser.
- **Scroll Animations**: Powered by GSAP and ScrollTrigger for buttery smooth pinning, scrubbing, and text staggering.
- **Smooth Scrolling**: Implements Lenis smooth scroll for a premium feel.
- **Responsive Layout**: Designed with flexbox and viewport units to scale beautifully.

## Setup
Simply open `index.html` in your browser, or host it using a local server (like Live Server in VS Code) to view the 3D model properly without CORS issues.

## Technologies Used
- HTML5, CSS3, JavaScript
- [Three.js](https://threejs.org/)
- [GSAP (ScrollTrigger)](https://greensock.com/gsap/)
- [Lenis](https://lenis.studiofreight.com/)
- [SplitType](https://github.com/lukePeavey/SplitType)
